# scFormTable 表格表单
前端工程师将面对各种各样的数据格式，业务也会有表单里含有对象数组的可能，进行封装后进行增删改操作，并且支持联动form表单的禁用属性

[演示地址](https://python-abc.xyz/scui-doc/demo/#/vab/formtable)
> 演示地址需模拟登录高权限角色后查看

## Props
|参数		|说明				|类型	|可选值	|默认值		|
|--			|--					|--		|--		|--			|
|v-model	|默认值				|Array	|-		|-			|
|addTemplate|新增行时数据模板		|Object	|-		|-			|
|placeholder|输入提示			|Array	|-		|暂无数据	|
